[−][src]Crate heap

The global allocator for the system. It starts off as a single fixed size allocator. When a more complex heap is set up, it is set as the default allocator.

Structs

Heap

The heap which is used as a global allocator for the system. It starts off with one basic fixed size allocator, the initial allocator. When a more complex heap is created and set as the DEFAULT_ALLOCATOR, then it is used.

Constants

HEAP_FLAGS

The heap mapped pages should be writable

Statics

Functions

init_single_heap

Initializes the single heap, which is the first heap used by the system.

set_allocator

Sets a new default allocator to be used by the global heap. It will start being used after this function is called.
