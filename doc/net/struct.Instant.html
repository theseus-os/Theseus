Expand description

A representation of an absolute time value.

The Instant type is a wrapper around a i64 value that represents a number of milliseconds, monotonically increasing since an arbitrary moment in time, such as system startup.

  • A value of 0 is inherently arbitrary.
  • A value less than 0 indicates a time before the starting point.
