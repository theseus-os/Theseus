pub fn create_application_namespace(
    recursive_namespace: Option<Arc<CrateNamespace>>
) -> Result<Arc<CrateNamespace>, &'static str>
Expand description

Create a new application CrateNamespace that uses the default application directory and is structured atop the given recursive_namespace. If no recursive_namespace is provided, the default initial kernel namespace will be used.

Return

The returned CrateNamespace will itself be empty, having no crates and no symbols in its map.
