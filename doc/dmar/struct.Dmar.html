Struct dmar::Dmar

source · 
pub struct Dmar<'t> { /* private fields */ }
Expand description

A wrapper around the DMAR ACPI table ([DmarReporting]), which contains details about IOMMU configuration.

You most likely care about the Dmar::iter() and Dmar::host_address_width() methods.
