A Theseus-specific port of Rust std’s fs module.

This moodule is a modified version of the “top-level” fs module file: [library/std/src/fs.rs]. We attempt to keep modifications to this as minimal as possible, in order to make it easier to integrate back into the real std lib later once we support that on Theseus.

Filesystem manipulation operations.

This module contains basic methods to manipulate the contents of the local filesystem. All methods in this module represent cross-platform filesystem operations. Extra platform-specific functionality can be found in the extension traits of std::os::$platform.
